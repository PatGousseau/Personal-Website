I'm a second year student at the University of British Columbia studying computer science. Currently learning web development and building projects on the side. Below are some of my favorite coding projects I developed.

Projects

Sentiment Analysis Twitter Bot

Twitter Bot that allows users to perform a sentiment analysis on their or any user’s profile upon their request. Plots a sentiment histogram of the user’s tweets and returns their most positive and negative tweets. Built in Java and Python using VADER, JavaFX and Twitter4J.


Instructions:
  • To analyze your own tweets: Mention the bot in any tweet (e.g "@yourStatsBot").
  • To analyze someone else's tweets: Mention the bot along with the other account (must be public) (e.g "@yourStatsBot @realDonaldTrump").

2D Arcade Game

2D game where the character avoids incoming bullets by jumping and ducking. Includes multiple levels of various difficulties and graphics, a leaderboard and power-ups. Built from scratch using only Java and the StdDraw library. Graphics created using Gimp.

Workout Tracker

Application that allows users to track and view statistics related to their fitness progress. Displays workout volume plot as well as the % improvement compared to the last session for each exercise to visualize progress. Built in Java using JSON for data persistence, JavaxSwing for the GUI and JFreeChart for plotting.
